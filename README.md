# 🏫✨ School Management System

> A Java-based Object-Oriented Programming (OOP) project for managing school operations like students, teachers, classes, and academic records efficiently.

---

## 📌 Overview 📖
The **School Management System** is designed using Java OOP principles to simulate real-world school operations.  
It helps manage student records, teacher details, class assignments, and performance tracking in a structured way.

---

## 🚀 Features ⚡
- 🎓 Add, update, and delete student records  
- 👨‍🏫 Manage teacher information   
- 📊 View student performance details  
- 🔍 Search and organize data efficiently  
- 💻 Simple console-based interface  

---

## 🛠️ Technologies Used 🧰
- ☕ Java (Core Java)
- 🧠 OOP Concepts:
  - Encapsulation 🔒
  - Inheritance 👨‍👩‍👧‍👦
  - Polymorphism 🔁
  - Abstraction 🎭
- 💻 IDE: Eclipse / IntelliJ IDEA
